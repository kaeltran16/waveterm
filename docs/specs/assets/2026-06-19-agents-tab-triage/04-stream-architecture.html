Open the stream only for a visible working agent. Collapse / scroll-away / exit → unmount → ctx cancels → goroutine + watcher tear down. No global refcounting.
vs. a WPS event
WPS broadcasts to all subscribers + needs a separate start/stop-watching control. Right for rare events (asks). Transcripts are per-agent + high-frequency → the stream's 1:1 lifecycle fits better.
